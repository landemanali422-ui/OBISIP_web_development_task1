Bean & Brew – Homepage (Short Explanation)

This webpage is the homepage of a café website called Bean & Brew, created using HTML and CSS.
The hero section has a full-screen coffee background, the café name, a tagline, and buttons for menu and contact.
The features section shows three key highlights: fresh coffee, cozy ambience, and quick service, displayed using cards with hover effects.
The footer contains contact details.
The website is responsive, visually attractive, and user-friendly
